# satellite-data-analysis
A website for Satellite Data Analysis using HTML and CSS.

Introduction
Satellite Data Analysis is the process of collecting, processing, interpreting, and analyzing data obtained from Earth-observing satellites. These satellites capture information about land, water, atmosphere, vegetation, and climate, enabling scientists and decision-makers to understand changes occurring on Earth.
Objectives
Monitor environmental changes.
Analyze land-use and land-cover patterns.
Support weather forecasting.
Improve agricultural productivity.
Detect and manage natural disasters.
Study climate change impacts.
Remote Sensing
Remote sensing is the science of obtaining information about an object or area without direct physical contact. Satellites use sensors to collect electromagnetic radiation reflected or emitted from Earth's surface.
Types of Remote Sensing
Passive Remote Sensing
Uses natural sunlight as the energy source.
Example: Optical satellite imagery.
Active Remote Sensing
Emits its own energy and measures the reflected signal.
Example: Radar and LiDAR systems.
Satellite Data Sources
NASA Landsat Program
European Space Agency Sentinel Missions
Indian Space Research Organisation Earth Observation Satellites
Data Processing Steps
1. Data Acquisition
Satellite imagery is collected from satellites orbiting Earth.
2. Preprocessing
Noise removal
Image correction
Geometric correction
Radiometric correction
3. Feature Extraction
Important features such as vegetation, water bodies, urban areas, and forests are identified.
4. Classification
Images are classified into categories using machine learning and image-processing techniques.
5. Analysis and Interpretation
The processed information is used for decision-making and scientific research.
